Code Standards

CODE128 vs ISO Grading

What it is

CODE128 — the format

A barcode symbology — defines how data is encoded as bars and spaces. It's one of many formats: EAN-13, QR, DataMatrix, etc. CODE128 describes what the code is.

What it measures

ISO/IEC — the quality grade

A quality standard (ISO/IEC 15416 for 1D, 15415 for 2D) that grades how well any barcode is rendered — contrast, quiet zone, modulation, edge sharpness — on a scale of A to F. ISO grading describes how well the code is displayed.

The relationship

Format ≠ Quality

A CODE128 barcode can score A (perfect) or F (failed) under ISO grading. In our test, the code was valid CODE128 — but the display quality was so poor it scored Grade 0.0 / F.

Stationary · High-end · 2D

REA VeriCube

5MP CMOS camera unit for verifying both 1D and 2D barcodes including QR codes. Supports 4 interchangeable camera modules (8mm–25mm). Includes DPM support.

DPM (Direct Part Marking) — codes physically engraved or etched onto surfaces, e.g. laser-marked parts. Harder to scan, requires specialized verification.

Battery-powered portable CCD scanner for 1D barcodes only. PC-free field use. Lower cost entry point for basic verification needs.

Assessment

REA VeriCube — Pros & Cons

✓ Advantages

  • ✓ ISO/IEC compliant — internationally recognized grading (A–F)
  • ✓ Detailed PDF reports with full parameter breakdown
  • ✓ Network-connected — centralized data management via TransWin32

✗ Disadvantages

  • ✗ Slow detection speed — not suitable for real-time or high-throughput scanning environments
  • ✗ High noise sensitivity — detects excessive noise, may flag codes that real POS scanners accept without issue
  • ✗ Dark mode not supported — cannot test barcodes displayed on dark-background screens
